pdsGraph utilise les bases de donnees crees par 'makePdsInfo.py' pour tenter de tracer les chemins qu'emprunte l'information dans le reseau de l'entreprise. Pour l'instant, l'affichage ce fait en ligne de commande sous la forme d'arborescence. Une affichage detaillee est aussi possible. 

Usage:
    ./makePdsGraph.py -p Produit [-td]
    
    '-p' : Produits affichees. Ecrites sous la forme de regex.
    '-t' : Affichage en arborescence.
    '-d' : Affichage detaillee.

Procedure pour ajouter un cluster dans 'makePdsGraph.py':
    1- Modifier le code pour ouvrir la nouvelle base de donnees.
    2- Ajouter a la liste des db pour la fonction 'findProducts()'.
    3- S'assurer de refermer la nouvelle base de donnees comme les autres.
